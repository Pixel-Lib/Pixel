uint32_t pxl::Timer::get_elapsed_time ( )

get_elapsed_time calculates the elapsed time since the timer was started in milliseconds.

Returns
the elapsed time in milliseconds since the timer was started. If the timer has not been started (when start_time is -1), it returns -1 to indicate that the timer is not running.

◆ get_time_left()

uint32_t pxl::Timer::get_time_left ( )

get_time_left calculates the time left based on the stop time set.

Returns
the time left in milliseconds until the stop time set in the stop_time variable. If the stop_time is not set (i.e., it is -1), the function returns -1 to indicate that the stop time is not set.
