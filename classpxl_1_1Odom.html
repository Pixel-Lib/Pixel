Detailed Description

The Odom class represents an odometry system that tracks the robot's position and orientation.

The Odom class uses a combination of tracking wheels and IMUs (Inertial Measurement Units) to calculate the robot's position and orientation on the field. It provides methods for initialization, calibration, and updating the odometry values.
