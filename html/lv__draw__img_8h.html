◆ lv_img_decoder_open_f_t

typedef const uint8_t*(* lv_img_decoder_open_f_t) (const void *src, const lv_style_t *style)

Open an image for decoding. Prepare it as it is required to read it later

Parameters
srcthe image source. Can be a pointer to a C array or a file name (Use lv_img_src_get_type to determine the type)
stylethe style of image (maybe it will be required to determine a color or something)
Returns
there are 3 possible return values: 1) buffer with the decoded image 2) if can decode the whole image NULL. decoder_read_line will be called to read the image line-by-line 3) LV_IMG_DECODER_OPEN_FAIL if the image format is unknown to the decoder or an error occurred
